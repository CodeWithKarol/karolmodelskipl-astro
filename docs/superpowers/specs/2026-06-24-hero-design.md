# Hero Section Design Specification

## Overview
A full-screen, impact-driven Hero section inspired by the "Mighty" style. Features a high-quality product screenshot background with a gradient overlay for optimal text legibility.

## Components
- **Background**: Full-viewport image/screenshot.
- **Overlay**: Dark gradient or semi-transparent overlay to ensure text contrast.
- **Content**: 
  - Central, large-scale headline (Bold, high contrast).
  - Clear, concise sub-headline.
  - Primary CTA button (distinct, high-visibility style).
- **Alignment**: Centered text (standard "Mighty" style).

## Aesthetic Direction
- **Style**: High-Impact, Bold.
- **Contrast**: High (White text on darkened background).
- **Legibility**: Guaranteed by the dark gradient overlay.

## Implementation Details
- **Tech Stack**: Astro component, Tailwind CSS.
- **Responsive**: Full-height viewport (`h-screen`), centered content via Flexbox/Grid.

## Success Criteria
- Section covers the full viewport height.
- Text is perfectly readable regardless of the background image details.
- Strong visual hierarchy.
