# Light Theme Hero Design Specification

## Overview
A full-screen, impact-driven Hero section with a refined light aesthetic. Features a soft gradient background surface and a product screenshot with depth, ensuring high readability and a professional enterprise feel.

## Components
- **Background**: Soft gradient surface (White to Very Light Gray).
- **Content**: 
  - Central, large-scale headline (Bold, deep black/navy).
  - Clear, concise sub-headline.
  - Primary CTA button (Strong color, high visibility against light background).
- **Visuals**: Product screenshot with a subtle shadow/depth effect to "pop" off the light background.

## Aesthetic Direction
- **Style**: Sophisticated Enterprise/Clean.
- **Contrast**: High (Dark text on light surface).
- **Feel**: Open, professional, airy.

## Implementation Details
- **Tech Stack**: Astro component, Tailwind CSS.
- **Background**: `bg-gradient-to-br from-white to-gray-50`.
- **Text**: `text-gray-900` for primary text.

## Success Criteria
- Section covers the full viewport height.
- Text is perfectly readable with high contrast.
- Visual hierarchy is maintained using bold typography and strategic color usage.
