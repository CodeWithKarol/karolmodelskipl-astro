# Studio Nika Style Footer Design Specification

## Overview
A hyper-minimalist, 4-column footer in the architectural "Studio Nika" grid style, displaying branding details, strategic contact information, solutions, and trust signals.

## Components
- **Layout**:
  - Desktop: 4 equal symmetrical columns divided by thin gray lines (`border-r border-gray-100`).
  - Mobile: Clean single-column layout with vertical margin-spacing.
- **Columns**:
  - Column 1: Brand description and "Model Elite" narrative.
  - Column 2: Compact address and Warszawie/Polska operational coverage.
  - Column 3: "Rozwiązania dla firm" navigation links.
  - Column 4: Trust signals, blog navigation, and active CTA links.
- **Bottom Bar**:
  - Horizontal separator line (`border-t border-gray-100`).
  - Contains NIP, REGON, and Karol Modelski copyright statement.

## Aesthetic Direction
- **Style**: Symmetrical Editorial Grid.
- **Borders**: Thin gray dividers, flat edges, no shadows, no rounded corners.
- **Colors**: Pure white background, charcoal/slate text for optimal professional contrast.

## Implementation Details
- **Tech Stack**: Astro component, Tailwind CSS.
