# Header Design Specification

## Overview
This design aims to replicate the "Bobbin" header navigation from One Page Love: a clean, editorial, and minimalist header.

## Components
- **Container**: Full-width, sticky at the top, white background with subtle shadow or thin border for separation.
- **Logo**: Text-based, bold weight, left-aligned.
- **Navigation Links**: Minimalist, clean sans-serif/serif font, evenly spaced, subtle hover effect (e.g., color shift or underline).
- **CTA**: Minimal outline button, refined hover state.

## Aesthetic Direction
- **Style**: Refined Minimalism
- **Typography**: Clean, professional typeface.
- **Colors**: Monochrome or high-contrast palette (Black/White).

## Implementation Details
- **Tech Stack**: Astro components.
- **Styling**: Tailwind CSS (as currently used in the project).
- **Responsiveness**: Standard responsive grid/flexbox for mobile/desktop.

## Success Criteria
- Header is sticky.
- Navigation links are clean and interactive.
- Design feels editorial and premium.
